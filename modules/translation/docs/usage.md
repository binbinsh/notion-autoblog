# translation Module

## Purpose

This module provides a reusable machine-translation notice for pages generated by `notion-autoblog`.

It supports two integration styles:

- partials for template-level integration
- a shortcode for manual insertion in Markdown content

## Import

Initialize Hugo Modules in your site and import this module:

```toml
[module]
  replacements = "github.com/binbinsh/notion-autoblog/modules/translation -> ../notion-autoblog/modules/translation"

  [[module.imports]]
    path = "github.com/binbinsh/notion-autoblog/modules/translation"
```

If your local directory is not named `notion-autoblog/`, update the replacement path accordingly.

## Partial Integration

Render the notice above your page content:

```go-html-template
{{ partial "translation/notice.html" . }}
```

The partial renders automatically when the page front matter includes:

- `notion_source_language`
- `notion_source_path`

## Shortcode Integration

You can also insert the notice manually in Markdown:

```md
{{< translation-note sourceLang="zh" sourcePath="posts/example.zh.md" >}}
```

If you omit the parameters, the shortcode falls back to the current page front matter.
