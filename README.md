# notion-autoblog

从 Notion 同步内容到 Hugo，并提供可直接复用的部署模板。

```bash
git submodule add https://github.com/binbinsh/notion-autoblog notion-autoblog
uv venv --python 3.10
uv pip install notion-autoblog/
uv run notion-autoblog --site-dir .
```
